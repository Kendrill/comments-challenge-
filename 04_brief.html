Задание для команды · Автоматический скрининг резюме

04 🚨 Ситуация
Кризис
Массовый отток сотрудников из «НоваТех». HR парализован. На критические вакансии Senior Full-Stack Developer приходит до 100 резюме в день в разных форматах (docx, pdf). Обработать вручную невозможно — нужна автоматизация через AI.
Ваша задача
Создать решение, которое пакетно обрабатывает резюме из папки (файлы .docx и .pdf) и с помощью AI-инструментов выдаёт идеально подходящих кандидатов.
Оценка: 1 балл за каждого правильно найденного. Можно получить дополнительный балл (или потерять, хе-хе).
📥 На вход
Архив-папка resumes.zip (файлы .docx и .pdf) + описание вакансии
⚙️ Инструмент
Любые AI-инструменты: ML-чат, агент в IDE, API — на ваш выбор
📤 На выход
Список идеальных кандидатов с номерами и обоснованием
🎯 Пилот: участник №4, назначенный фасилитатором
Что у вас есть
  • Архив-папка resumes.zip — резюме в файлах .docx и .pdf (разные форматы и структуры)
  • Описание вакансии — station4_vacancy.html
  • AI-инструменты — ML-чат, агент в IDE, подключение по API — используйте то, что считаете нужным
  • Компьютер — Python, IDE, браузер, терминал
⚠️Резюме приходят в разных форматах файлов (.docx, .pdf) и в разных структурах (разный порядок блоков, разная полнота данных). Ваш инструмент должен справляться со всеми вариантами.
📝 Алгоритм работы
Рекомендуемые шаги (20 минут)
  1. Обсудите подход (2–3 мин) — Как парсить .docx и .pdf? Как обработать все файлы пачкой? Можно распараллелить работу в команде.
  2. Парсинг резюме и формирование JSON (7 мин) — Распакуйте архив, с помощью промта напишите скрипт, способный извлечь текст из всех документов. Сформируйте 1 или несколько JSON'ов.
  3. Формирование мастер-промта (5 мин) — Передайте описание вакансии в AI-инструмент для формирования мастер-промта с требованиями к позиции, результат можно также JSON.
  4. Запрос результата, анализ правильности (3 мин) — С помощью AI-инструмента проанализируйте JSON резюме и требования к вакансии (мастер-промт) для поиска идеальных кандидатов. Убедитесь, что анализ прошел верно (к примеру проверьте основные критерии типа 5+ лет опыта и т.д.).
  5. Подготовка к демо (2 мин) — Сформируйте ответ: кандидаты с номерами и обоснованием.
Распределение времени (30 мин)
5 мин — вводная
20 мин — работа
5 мин — демо
Бонусное задание
Если основное задание выполнено, можно заработать дополнительный балл. Фасилитатор выдаст resumes_prep_day.zip — ещё один набор резюме. Прогоните их через свой скрипт.
+1 балл — за всех правильно найденных идеальных кандидатов из этого набора.
−1 балл — за каждого ошибочно выбранного кандидата из этого набора.
🎤 Что показать на демо (5 мин)
Подготовьтесь презентовать
  1. Архитектура решения — как парсите файлы? какой подход к анализу? (30 сек)
  2. Живой запуск — запустите скрипт и покажите процесс обработки (2 мин)
  3. Результат — идеальные кандидаты с номерами и обоснованием (1 мин)
  4. Инсайт — что узнали, что бы доработали при наличии времени (30 сек)
